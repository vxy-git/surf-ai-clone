## 🐛 已知限制

### 1. Etherscan 仅支持 ETH
**原因**: 其他链需要不同的 API (BSC Scan, Polygon Scan等)

**解决方案**: 可以添加多链支持
```typescript
if (symbol === 'BNB') {
  // 使用 BSC Scan API
} else if (symbol === 'MATIC') {
  // 使用 Polygon Scan API
}
```

### 2. LunarCrush API 速率限制
**原因**: 取决于您的套餐

**解决方案**: 添加请求缓存

### 3. CoinGecko 免费版限制
**原因**: 50次/分钟

**解决方案**:
- 添加 Redis 缓存
- 升级到专业版
